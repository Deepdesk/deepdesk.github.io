# StickyMessage

### Overview

```ts
/**
 * StickyMessage
 */
type StickyMessage = {
    /**
     * The title of the sticky message
     */
    title: string;

    /**
     * The message of the sticky message
     */
    message: string;

    /**
     * An optional image for in the sticky message
     */
    image: string;
};
```
